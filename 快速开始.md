# 🚀 快速开始指南

## 立即使用

### 方式一：双击打开（最简单）

1. 找到 `index.html` 文件
2. 双击打开
3. 开始使用！

### 方式二：本地服务器

```bash
# 进入项目目录
cd d:\po2

# 使用Python启动
python -m http.server 8000

# 或使用Node.js
npx http-server

# 浏览器访问
# http://localhost:8000
```

## 主要功能

### 1️⃣ 搜索功能

在搜索框输入中文或英文关键词：
- 输入"火球" → 查找所有包含火球的条目
- 输入"Fire" → 查找所有包含Fire的条目
- 输入"技能" → 查找所有技能相关内容

### 2️⃣ 分类筛选

点击分类按钮快速筛选：
- **技能** - 931条（主动技能、辅助技能）
- **装备** - 619条（传奇装备、珠宝等）
- **货币** - 162条（通货、精华、催化剂）
- **词条** - 1162条（天赋、被动技能）
- **机制** - 1048条（游戏关键词、区域）
- **怪物** - 147条（怪物、Boss）

### 3️⃣ 组合使用

搜索 + 筛选可以组合使用：
1. 先选择"技能"分类
2. 再搜索"火焰"
3. 得到所有火焰相关的技能

### 4️⃣ 清除筛选

点击"清除筛选"按钮，恢复显示所有数据。

## 更新数据

如果游戏更新了新内容，运行：

```bash
node parse_combined.js
```

脚本会自动：
1. 下载最新官方数据
2. 匹配中英文
3. 更新 `data.js` 文件
4. 刷新页面即可看到新数据
