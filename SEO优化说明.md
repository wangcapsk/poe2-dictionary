## 📋 部署前需要修改的内容

### 🔧 必须修改的URL
在以下文件中，将 `https://yourwebsite.com/` 替换为你的实际域名：

1. **index_multilang.html**
   ```html
   <!-- 第33行 -->
   <meta property="og:url" content="https://你的域名.com/">
   
   <!-- 第34行 -->
   <meta property="og:title" content="...">
   
   <!-- 第35-36行 -->
   <meta property="og:image" content="https://你的域名.com/og-image.jpg">
   
   <!-- 第39-42行 Twitter卡片 -->
   <meta property="twitter:url" content="https://你的域名.com/">
   <meta property="twitter:image" content="https://你的域名.com/og-image.jpg">
   
   <!-- 第46行 Canonical链接 -->
   <link rel="canonical" href="https://你的域名.com/">
   
   <!-- 第58行 Schema.org数据 -->
   "url": "https://你的域名.com/"
   ```

2. **robots.txt**
   ```
   Sitemap: https://你的域名.com/sitemap.xml
   ```

3. **sitemap.xml**
   ```xml
   <loc>https://你的域名.com/</loc>
   <loc>https://你的域名.com/index_multilang.html</loc>
   ```
