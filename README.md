## 📖 项目简介

这是一个**流放之路2 (Path of Exile 2)** 的中英文对照查询工具，收录了 **4000+** 条游戏术语翻译，包括：

- ⚔️ **技能系统**：技能宝石、辅助宝石、元技能宝石
- 🛡️ **装备道具**：传奇装备、永恒珠宝、裂界碎片、巅峰钥匙等
- 💎 **货币系统**：堆叠货币、增幅器、金库钥匙、地图碎片等
- 📜 **天赋词条**：被动天赋、升华职业
- ⚙️ **游戏机制**：关键词、世界区域、预兆、咒符、引路石等
- 👹 **怪物生物**：怪物名称、首领Boss

## ✨ 功能特性

- 🔍 **智能搜索**：支持中英文双向查询
- 🏷️ **分类筛选**：6大类别快速定位
- 🌐 **多语言支持**：简体中文、繁體中文、English
- 📱 **响应式设计**：完美适配移动端和桌面端
- ⚡ **性能优化**：本地数据，秒级响应
- 🎨 **现代UI**：暗色主题，舒适视觉体验

## 🎯 使用说明

1. **搜索查询**：在搜索框输入中文或英文关键词
2. **分类筛选**：点击分类按钮快速过滤
3. **繁体切换**：勾选"显示繁體中文"查看繁体翻译
4. **清除筛选**：点击"清除筛选"重置所有条件
