# 🚀 从这里开始

## 快速使用

### 1️⃣ 打开网站（推荐）

**单语言版**（简体中文 ↔ 英文）
```
双击打开: index.html
```

**多语言版**（简体 / 繁体 / 英文）🆕
```
双击打开: index_multilang.html
```

### 2️⃣ 开始使用
- 在搜索框输入关键词
- 点击分类按钮筛选
- 多语言版可切换语言

就这么简单！ ✨
